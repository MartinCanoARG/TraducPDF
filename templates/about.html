Sobre TraducPDF

Inicio Historial Sobre esta app

TraducPDF es una aplicación creada por Martin Cano como proyecto final para el curso CS50x 2025.

Permite subir archivos PDF, traducir su contenido automáticamente a otro idioma y descargar la versión traducida en segundos.

La aplicación está pensada para ayudar a estudiantes universitarios en carreras como medicina o derecho a comprender textos académicos en otros idiomas.

Este proyecto utiliza Python, Flask, pdfplumber, reportlab y la API de OpenAI.
